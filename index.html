Gmail Label & Filter MCP

A private, locally-run tool for safely organizing one Gmail account — its labels, filters, and inbox rules.

Gmail Label & Filter MCP is a personal Model Context Protocol server that its author runs on their own computer to manage their own Gmail account. It is not a hosted service, has no other users, and stores no data on any server.

What it does

  • List, create, update, and delete Gmail labels
  • List, create, replace, and delete Gmail filters
  • Apply and remove labels on messages and threads
  • Preview and apply semantic inbox rules

How it uses Google data

It requests only the minimum Gmail scopes needed to manage labels, filters, and settings:

  • gmail.labels — read and manage labels
  • gmail.modify — apply/remove labels on mail
  • gmail.settings.basic — read and manage filters

It does not request full-mailbox access (https://mail.google.com/) and does not read message bodies — only metadata and short snippets needed to preview a rule.

Safety

  • Mutating actions default to a dry-run preview
  • Destructive actions require an explicit, single-use confirmation token
  • OAuth tokens are stored encrypted on the author's own machine
See the Privacy Policy for how data is handled.

Personal MCP Tools

A private collection of locally-run Model Context Protocol servers the author uses to work with their own Google account.

Personal MCP Tools is a set of personal Model Context Protocol servers that their author runs on their own computer, for their own single Google account. It is not a hosted service, has no other users, and stores no data on any server.

Tools in this collection

Gmail Label & Filter MCP

Safely manage the author's Gmail labels, filters, and inbox rules — no reading of message bodies.

Additional personal tools for other Google services may be added over time; each requests only the minimum scopes it needs.

How it uses Google data

Each tool requests only the minimum Google scopes needed for its purpose. The scopes currently in use are:

  • gmail.labels — read and manage Gmail labels
  • gmail.modify — apply/remove labels on mail
  • gmail.settings.basic — read and manage Gmail filters

These tools do not request full-mailbox access (https://mail.google.com/) and do not read message bodies — only the metadata and short snippets needed to preview an action.

Safety

  • Mutating actions default to a dry-run preview
  • Destructive actions require an explicit, single-use confirmation token
  • OAuth tokens are stored encrypted on the author's own machine
See the Privacy Policy for how data is handled.

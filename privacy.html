Privacy Policy

Gmail Label & Filter MCP · Last updated 10 July 2026

Gmail Label & Filter MCP (“the app”) is a personal, open-source Model Context Protocol server run by its author on their own computer to manage their own Gmail account. This policy explains how it handles Google user data.

Who this covers

The app is a single-user, self-hosted tool. It has no other users, no backend service, and no shared infrastructure. The only Google account it accesses is the author's own.

Google data the app accesses

Using the following Gmail OAuth scopes, the app accesses only data needed to manage labels, filters, and inbox rules:

  • gmail.labels — Gmail labels
  • gmail.modify — applying and removing labels on messages and threads
  • gmail.settings.basic — Gmail filters and basic settings

The app does not request full-mailbox access and does not read the content of email messages. It reads only message metadata and short snippets when needed to preview the effect of a rule.

How data is used

Data is used solely to perform the label, filter, and rule operations the author explicitly requests. It is not used for advertising, analytics, profiling, or any secondary purpose.

Storage and sharing

  • OAuth tokens are stored encrypted on the author's local machine and never transmitted anywhere except directly to Google's official APIs.
  • No Gmail data is stored on any server or shared with any third party.
  • The app does not sell, rent, or transfer any Google user data. Its use of Google data complies with the Google API Services User Data Policy, including the Limited Use requirements.

Data retention and deletion

The app keeps no copy of Gmail content. The only persisted item is the encrypted OAuth token on the author's computer, which can be deleted at any time by removing the local token file, and access can be revoked at myaccount.google.com/permissions.

Contact

This is a personal, single-user application. Questions about this policy can be raised through the project's GitHub issues page. Access can be revoked at any time at myaccount.google.com/permissions.
